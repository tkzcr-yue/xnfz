项目特色Project features

  • 项目特色一

    Film base in chengdu in 2014

    1.构建了“三线三面”的立体虚拟仿真教学系统。

    本项目通过“展会参观-工厂参观-商务洽谈”三条完整的采购路线及“陶瓷-旗袍-智能护理机器人”三类产品提供了一整套可操作的语言情景，让留学生真实体验“广交会”采购全过程，使得留学生商务汉语实践训练突破了时空和商务私密性的局限。

  • 项目特色二

    Film base in chengdu in 2014

    全方面融入了商务汉语知识、技能和文化内容的学习。

    实验通过多个模块、中英双语，承担商务汉语不同知识点的教学任务，学习者可自由组合学习模块，使得商务文化可感知、商务知识可操作、商务技能可验证，提升了商务汉语学习的探究性和高阶性。

  • 项目特色三

    Film base in chengdu in 2014

    实现了“思政、教学、科研”有机结合。

    系统在合同洽谈模块设计了差异性谈判策略，不同策略契合相异操作步骤，同时融合了人工智能、大数据交互技术，收集汇总学习数据，分析商务汉语应用的痛点和弱点，针对性地调整教学模式，进一步提升留学生汉语学习的内驱力。
